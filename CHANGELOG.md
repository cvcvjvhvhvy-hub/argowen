# Changelog

جميع التغييرات المهمة في هذا المشروع سيتم توثيقها في هذا الملف.

## [1.0.0] - 2024-01-XX

### إضافات
- نظام المصادقة بالهاتف باستخدام Firebase Auth
- إدارة المستخدمين (بقالة، تاجر، مدير)
- إنشاء وإدارة الفواتير
- نظام المناقصات للتجار
- واجهة مستخدم متجاوبة
- دعم اللغة العربية

### إصلاحات
- إصلاح مشاكل null safety في جميع النماذج
- تحسين معالجة الأخطاء في شاشات المصادقة
- إضافة التحقق من صحة البيانات المدخلة
- إصلاح أسماء الكلاسات لتتبع أفضل الممارسات

### تحسينات
- تحسين أداء التطبيق
- تحسين تجربة المستخدم
- إضافة رسائل خطأ واضحة
- تحسين هيكل الكود

### أمان
- تطبيق قواعد أمان Firebase
- التحقق من صحة البيانات
- حماية من null pointer exceptions
